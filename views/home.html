Welcome to the showcase of the official Authentication plugin for rable.js!

rable.js-auth is written by the same developer that made rable.js, and thereby the official authentication plugin.
The plugin makes use of both rable.js and rable.js-api and provides easy user authentication. It registers a middleware which checks if the request is authenticated, provides a login route and a route to check if a client is signed in.

For this showcase you can login with the two already configured users stored in a locdb database so that everyone can easily try it out on their own computer.

John Doe: johndoe@email.com / SamplePassword
Mary Doe: marydoe@email.com / AnotherPassword

In this case, their email address excluding "@email.com" also acts as their username. You can login with both their username and email address.

For further information about the mechanics of how the Library works, please visit the docs (also publicly available here).
